# Traveling salesman with Ant Colony Optimisation Algorithm

## *English*

_Objectives_

Formulate problems as search problems and identify ways to solve them based on evolutionary algorithms. Specification, design and implementation of search methods based on Ant Colony Optimization (ACO) algorithms.

_Theoretical aspects_

Problem solving as an optimization process
Types of optimization problems.
Ways to solve search problems -> Identify the optimal potential solution:
- Determining the components of the problem
o Conditions (constraints) that the solution must satisfy (partially or totally)
o Function for evaluating a potential solution -> identifying the optimal
- Defining the search space
Establishing the strategy for identifying the optimal solution in the search space


_Requirements_

Specify, design, and implement an application that solves the travel agent problem using the ACO technique. The application must allow:
- Upload problem data
- Choosing and parameterizing the method of solving the problem
- Display of the identified solution
- Specifying the performances of the chosen solving method
The application must comply with the specifications for input data and output data.

## *Romanian*

_Obiective_

Formularea problemelor ca probleme de căutare şi identificarea modalităţilor de rezolvare a lor
bazate pe algoritmi evolutivi. Specificarea, proiectarea şi implementarea metodelor de căutare
bazate pe algoritmi de tip Ant Colony Optimisation (ACO). 

_Aspecte teoretice_

Rezolvarea problemelor ca proces de optimizare
Tipuri de probleme de optimizare.
Modalitati de rezolvare a problemelor de căutare -> Identificarea soluţiei potenţiale optime:
- Stabilirea componentelor problemei
o Condiţii (constrângeri) pe care trebuie să le satisfacă (parţial sau total) soluţia
o Funcţie de evaluare a unei soluţii potenţiale -> identificareaa optimului
- Definirea spaţiul de căutare
Stabilirea strategiei de identificare a soluţiei optime în spaţiul de căutare 


_Cerinţe_

Specificaţi, proiectaţi şi implementaţi o aplicaţie care să rezolve problema comisului voiajor folosind tehnica ACO. Aplicaţia trebuie să permită:
- Încărcarea datelor problemei
- Alegerea şi parametrizarea metodei de rezolvare a problemei
- Afişarea soluţiei identificate
- Precizarea performanţelor metodei de rezolvare alese
Aplicația trebuie să respecte specificațiile privind datele de intrare și datele de ieșire.
